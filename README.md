# Shuting_game

횡스크롤 슈팅 게임입니다.

# 사용 중인 유니티 버전
2019.4.20f1

# 기능

1. 랜덤으로 적을 생성합니다.
2. 총 3종류의 적이 있습니다.
3. 대미지 증가, 필살기, 골드 총 3가지의 아이탬이 있습니다.
4. 스킬을 쓰면 적을 모두 죽입니다.
5. 적의 공격을 총 3번 받으면 게임 종료 화면이 나오게 됩니다.
